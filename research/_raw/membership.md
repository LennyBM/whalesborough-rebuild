# Membership

The W Club is a great place to be, which is why there is currently a waiting list for all of our membership types. If you’d like to be first in line to snap up a membership as and when they become available make sure you’re on the list by emailing [wclub@whalesborough.co.uk](mailto:wclub@whalesborough.co.u).

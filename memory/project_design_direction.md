The approved design direction is **Option 2: Coastal Editorial** from Google Stitch 2.0.

Fonts: Newsreader (serif, headlines) + Plus Jakarta Sans (sans-serif, body)
Primary colour (CTAs): #703a1d (cognac)
Secondary colour (nav/structure): #4a6457 (deep sage)
Background: #fbf9f6 (warm white)
0px border radius throughout — sharp corners, editorial feel
No dark mode — light only
No 1px borders — tonal shifts only

Design files saved at: `design-system/` folder in the project root
Design system reference: `design-system/DESIGN-SYSTEM-REFERENCE.md`

**Why:** Client chose this from 5 Stitch options. Matches the lovable.app reference aesthetic. Condé Nast Traveller meets luxury travel editorial.
**How to apply:** Every new page built should follow DESIGN-SYSTEM-REFERENCE.md exactly.

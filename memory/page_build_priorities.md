## Build Priority Order

Based on briefing doc synthesis + existing files in the project.

---

### TIER 1 — High impact, needs immediate attention

**1. Homepage (`/index.html`) — EXISTS, needs upgrading**
- Add seasonal vibe switcher module
- Ambient hero video (hero.mp4 exists — ensure it conveys estate "circadian rhythm")
- Integrate availability search bar into/below hero
- Add 500-acre interactive estate map section
- Social proof section: TripAdvisor/Google ratings inline, award badges (assets exist in /awards/)
- Lead magnet block (AI Itinerary Planner or destination guide CTA)
- Availability calendar lower on page
- UX hierarchy: Aspiration → Rooms → Amenities → Social Proof → Book

**2. Cottages Overview (`/our-cottages/index.html`) — EXISTS, needs upgrading**
- Real-time AJAX filtering (no page reloads) by: guests, pets, hot tub, EV charger, step-free, bedrooms
- Persona filter presets: "Families", "Couples", "Dog Friendly", "Large Groups", "Step-Free"
- All-in transparent pricing from first card result
- Comparison capability across units
- "Book Direct = Best Rate" messaging

**3. Spa & Treatments (`/spa-and-treatments/index.html`) — EXISTS, needs upgrading**
- Spa menu redesigned as "digital magazine" — treatments as Rituals/Journeys
- "Estate-to-Treatment" section: which botanicals come from the farm, used in which rituals
- Editorial illustrations or high-end photography for each treatment category
- Sticky booking/enquiry bar

### TIER 2 — Important, some foundation exists

**4. Dog-Friendly Hub (`/dog-friendly-breaks/index.html`) — EXISTS, needs upgrading**
- This is a high-intent SEO silo: `/dog-friendly-cornwall/`
- Sections: Pooch Welcome Pack, on-site dog showers, enclosed gardens, EV charging
- Year-round vs seasonal beach restrictions map/table
- Dog-friendly local walks + beaches guide (feeds into local area map)
- Filterable cottage grid: only dog-friendly units

**5. Sustainability Page (`/green-sustainability-credentials/index.html`) — EXISTS, needs upgrading**
- Replace vague claims with Live Sustainability Dashboard visual
- Real data: energy production, rewilding acres, biodiversity audit results
- "Eco-Champion" positioning: specific numbers and verified credentials
- Sustainability poster asset exists: `/sustainability/sustainability-poster.jpg`

**6. Individual Cottage Pages — PARTIAL (some built, some missing)**
- Every page needs: BLUF opening paragraph, schema-ready structure, amenity table, VacationRental markup
- All-in pricing transparency
- Single-unit availability calendar
- "Book This Room" CTA within gallery
- 360° tour placeholder (or note as future asset)
- ContainedInPlace link back to main resort entity

### TIER 3 — New pages to build

**7. Weir Restaurant (`/weir-restaurant/index.html`) — EXISTS, check quality**
